# Hexlet Webinars

Материалы к вебинарам и воркшопам для студентов Хекслета.

В репозитории собраны презентации, примеры кода, SQL-файлы, учебные базы данных и дополнительные материалы, которые можно использовать во время занятия или после него для самостоятельной практики.

## Как устроен репозиторий

Каждый вебинар лежит в отдельной папке.

Обычно внутри папки есть:

- `README.md` — описание конкретного вебинара;
- ссылка на презентацию;
- примеры кода или SQL-запросов;
- файлы, которые могут понадобиться во время занятия;
- дополнительные материалы для самостоятельного разбора.

Практические задания не вынесены в отдельный общий раздел: всё, что относится к конкретному вебинару, лежит в папке этого вебинара.

## Вебинары

| Папка | Тема | Краткое описание |
|-------|------|------------------|
| [`1_python_functools`](./1_python_functools) | Python: `functools` | Введение в инструменты стандартной библиотеки для работы с функциями: декораторы, кэширование, частичное применение аргументов и другие приёмы функционального стиля в Python. |
| [`1_python_itertools`](./1_python_itertools) | Python: `itertools` | Разбор модуля `itertools`: работа с итераторами, объединение последовательностей, срезы потоков, группировка, комбинации и генерация данных без лишних промежуточных списков. |
| [`2_python_functools`](./2_python_functools) | Python: `functools` | Практический воркшоп по `functools`: как писать корректные декораторы, использовать `wraps`, `cache`, `lru_cache`, `partial`, `reduce` и `singledispatch`. |
| [`3_python_functools`](./3_python_functools) | Python: `functools` | Ещё одна версия воркшопа по `functools` с акцентом на практическое применение: декораторы, кэширование, функции как объекты и выбор поведения по типам. |
| [`sql_for_dev_and_ops`](./sql_for_dev_and_ops) | SQL для разработчиков и DevOps | Вебинар по основам SQL на SQLite-базе с супергероями. Разбираются `SELECT`, `WHERE`, `ORDER BY`, `LIMIT`, `INSERT`, `UPDATE`, `DELETE`, `FOREIGN KEY`, `JOIN`, `COUNT` и `GROUP BY`. |

## Для кого эти материалы

Материалы рассчитаны в первую очередь на студентов начальных модулей и начинающих разработчиков, которые хотят:

- закрепить темы после вебинара;
- повторить примеры самостоятельно;
- открыть презентацию и пройти материал в своём темпе;
- запустить готовые файлы локально;
- использовать материалы как шпаргалку.

## Как пользоваться

1. Откройте папку нужного вебинара.
2. Прочитайте локальный `README.md`.
3. Откройте презентацию по ссылке.
4. Запустите примеры из папки, если они есть.
5. Используйте дополнительные файлы для самостоятельной практики.

## Требования

Точные требования зависят от конкретного вебинара.

Например:

- для Python-вебинаров нужен установленный Python;
- для SQL-вебинара может понадобиться SQLite;
- отдельные зависимости описаны в `README.md` соответствующей папки.

## Статус

Репозиторий развивается постепенно. Материалы могут обновляться, дополняться примерами, практикой, пояснениями и исправлениями после проведения вебинаров.
